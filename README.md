# Title
This repository contains bioinformatics analysis scripts and R code for visualization, developed for the manuscript "Identification of ISG+ Aged Neutrophil as a Key Immune Driver for Checkpoint Inhibitor Pneumonitis and Idiopathic Pulmonary Fibrosis". Sherlock-_Lung_ is a comprehensive study that aims to trace lung cancer etiology in never smokers by analyzing genomic data in tumor and surrounding lung tissue.
