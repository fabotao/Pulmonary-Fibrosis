# Pulmonary-Fibrosis
This repository contains bioinformatics analysis scripts and R code for visualization, developed for the manuscript "Identification of ISG+ Aged Neutrophil as a Key Immune Driver for Checkpoint Inhibitor Pneumonitis and Idiopathic Pulmonary Fibrosis". 

## Datasets
Input datasets can be download from [zenodo](https://zenodo.org/records/18044743/files/processed_data.rar?download=1)
